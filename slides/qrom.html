Quantum Random Oracle Model
Allow superposition queries to the random oracle.
Rough sketch:
Initial state: $D=\ket{\bot}$
First query: $\ket{x_1}\ket{0}+\ket{x_2}\ket{0}$.

Oracle replies $\ket{x_1}\ket{y_1}+\ket{x_2}\ket{y_2}$

$\Rightarrow D=\sum_{y_1}\ket{[(x_1, y_1)]} + \sum_{y_2}\ket{[(x_2, y_2)]}$

Second query: $\ket{x_3}\ket{0}+\ket{x_4}\ket{0}$.

$\Rightarrow D= \sum_{y_1, y_3}\ket{[(x_1, y_1), (x_3, y_3)]} + \sum_{y_1, y_4}\ket{[(x_1, y_1), (x_4, y_4)]}$

$\qquad + \sum_{y_2, y_3}\ket{[(x_2, y_2), (x_3, y_3)]} + \sum_{y_2, y_4}\ket{[(x_2, y_2), (x_4, y_4)]} $
