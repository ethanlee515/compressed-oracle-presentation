Database Transitions
Define the quantum transition capacity as $$[\![P\xrightarrow{k}P']\!]:=\max_{x, y} ||P'cO_{xy}P||$$ which measures how likely it is that a database that satisfies $P$ turns into a database that satisfies $P'$ from a query.
Define $$[\![P\xrightarrow{k,q}P']\!]:=\sup_{U_i} ||P' cO^k U_{q-1} cO^k\ldots U_1 cO^k P||$$ which measures how likely it is that a database that satisfies $P$ turns into one that satisfies $P'$ from a quantum algorithm.
Quick exercise: For any sequence of properties $P_i$, $$[\![\neg P_0\xrightarrow{k, q}P_q]\!]\leq\sum_{i}[\![\neg P_{i-1}\rightarrow P_i]\!]$$
