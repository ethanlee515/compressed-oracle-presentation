More on Transitions Capacities
We can break down transitions using database properties.
A transition $\neg\mathsf{P}\rightarrow\mathsf{P}'$ is strongly recognizable by $\set{\mathsf{L}_i}$ if and only if $\mathsf{P}'\subseteq \cup_i\set{\mathsf{L}_i}\subseteq\mathsf{P}$.
Easier to think about the $\mathsf{P}=\mathsf{P}'$ case:
We have $\mathsf{P}=\cup_i\set{\mathsf{L}_i}$.
For any $D$ to satisfy $\mathsf{P}$, it is necessary and sufficient that $D$ satisfies one of the $\mathsf{L}_i$'s.
A transition $\mathsf{P}\rightarrow\mathsf{P}'$ is weakly recognizable by $\set{\mathsf{L}_i}$ if and only if $$ D\in\mathsf{P}\wedge D'\in\mathsf{P}'\Rightarrow \exists\mathsf{L}_i: D'\in\mathsf{L}_i \wedge (\exists x\in Supp(\mathsf{L}_i), D(x)\ne D'(x)) $$
There are non-uniform versions too.
