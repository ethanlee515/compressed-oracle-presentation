More on Transitions Capacities
We can also characterize transitions using database properties.
A transition $\neg P\rightarrow P'$ is strongly recognizable by $\set{L_i}$ if $P'\subseteq \cup_i\set{L_i}\subseteq P$.
Easier to think about the $P=P'$ case
We have $P=\cup_i\set{L_i}$.
For any $D$ to satisfy $P$, it is necessary and sufficient that $D$ satisfies one of the properties.
